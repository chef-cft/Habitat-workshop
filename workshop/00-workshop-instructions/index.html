What is an edge device?

An edge device is a computer from any kind - desktop, server, single board computer or IoT device, running in the "field" and not in the cloud. Point of Sale systems, invenvtory management systems, and Kiosks are common edge devices.

In addition smart TV's, as well as factory equipment, smart sensors, city road cameras and even vending machines are classifications of Edge Devices.

What is Progress Chef Edge Management?

Progress Chef Edge Management consists of a suite of interegrated services, tools and features that let allow you to keep your edge devices fleet host and applications updated, compliant, stable, and secure.

Progress Chef Edge Management allows you to:

  • Deploy and rollback software updates
  • Device/Host configuration managements
  • Device/Host monitor managements
  • Device/Host conpliance managements
  • Application conpliance managements
  • Remote device management/control
